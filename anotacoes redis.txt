
Redis - banco de dados em memória

- Usado para cache de informações
- Significado: REmote DIctionary Server.


Documentação do Redis: redis.io

- A instalação do Redis é feita apenas em sistemas baseados em Unix (linux, macOs)
- Para instalar no windows, é necessário habilitar o WSL e adicionar uma instância do Linux,
  significando que, apesar de termos uma máquina windows, ainda assim estaríamos trabalhando 
  com Linux.

- Solução para as aulas, e posteriormente para futuras implementações:
 
	criar uma conta na nuvem para uso do Redis: redis.com

Implementação de chaves

	- Usar uma estrutura hierárquica, separada por : (dois pontos)

	Exemplos:

	cliente:c1234:nome
	cliente:c1234:email

	pedido:123/2023:total
	....

   Pesquisando chaves

	KEYS <padrao>

	KEYS *			retorna todas as chaves
	KEYS turma:*		retorna as chaves iniciadas por 'turma'
	KEYS *:cliente		retorna as chaves terminadas por 'cliente'




	empresa: escola
	empresa:escola
